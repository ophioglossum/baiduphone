# baiduphone

#### 百度关键词搜索手机号
##### 项目功能
- 输入关键词查询手机号，记录获取的来源网址信息
##### 优点
- 通过selenium绕过百度安全监测，该chromedriver驱动使用版本为91.0.4472.19.
- 使用QT作为界面，操作简单便捷
- 该项目使用pyqt5，selenium，multiprocessing，sqlalchemy，openpyxl模块
